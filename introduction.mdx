    Quick Start
  </h2>
  <p className="text-base text-gray-600 dark:text-gray-400 mb-8">
    Get your riding school API up and running in minutes
  </p>
  
  <Steps>
    <Step title="Clone and build the project">
      Clone the repository and build with Maven:
      
      ```bash
      git clone https://github.com/luckidetrenque/Backend-Proyecto-HRS.git
      cd Backend-Proyecto-HRS/hrs
      mvn clean package
      ```
    </Step>
    
    <Step title="Configure the database">
      Set up your database connection in `application.properties`. The API supports PostgreSQL, H2, and MySQL:
      
      ```properties
      spring.datasource.url=jdbc:postgresql://localhost:5432/hrs_db
      spring.datasource.username=your_username
      spring.datasource.password=your_password
      ```
      
      <Info>H2 in-memory database is configured by default for quick testing</Info>
    </Step>
    
    <Step title="Run the application">
      Start the Spring Boot application:
      
      ```bash
      mvn spring-boot:run
      ```
      
      The API will be available at `http://localhost:8080`
    </Step>
    
    <Step title="Authenticate and make your first request">
      Register a user and authenticate to get started:
      
      ```bash
      # Register a new user
      curl -X POST http://localhost:8080/api/v1/auth/register \
        -H "Content-Type: application/json" \
        -d '{
          "username": "admin",
          "email": "admin@hrs.com",
          "password": "SecurePass123",
          "rol": "ADMIN"
        }'
      
      # Login with Basic Auth
      curl -X POST http://localhost:8080/api/v1/auth/login \
        -u admin:SecurePass123
      ```
      
      <Accordion title="Example response">
        ```json
        {
          "id": 1,
          "username": "admin",
          "email": "admin@hrs.com",
          "rol": "ADMIN"
        }
        ```
      </Accordion>
    </Step>
  </Steps>
</div>
