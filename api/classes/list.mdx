Endpoints for retrieving riding class information from the HRS system.

## List All Classes

<api method="GET" endpoint="/api/v1/clases" />

Retrieves all classes without related entity details (basic information only).

### Response

Returns an array of `Clase` objects.

<ResponseField name="id" type="Long">
Unique identifier for the class
</ResponseField>

<ResponseField name="especialidad" type="Especialidad" required>
Class specialty. Valid values:
- `EQUINOTERAPIA` - Equine therapy
- `EQUITACION` - Horse riding
- `ADIESTRAMIENTO` - Training
- `MONTA` - Mounting
</ResponseField>

<ResponseField name="dia" type="LocalDate" required>
Class date in ISO-8601 format (YYYY-MM-DD)
</ResponseField>

<ResponseField name="hora" type="LocalTime" required>
Class time in ISO-8601 format (HH:mm:ss)
</ResponseField>

<ResponseField name="duracion" type="Integer" required>
Class duration in minutes (minimum 30)
</ResponseField>

<ResponseField name="estado" type="Estado" required>
Class state. Valid values:
- `PROGRAMADA` - Scheduled
- `INICIADA` - Started
- `COMPLETADA` - Completed
- `CANCELADA` - Cancelled
- `ACA` - Absent with notice (Ausente Con Aviso)
- `ASA` - Absent without notice (Ausente Sin Aviso)
</ResponseField>

<ResponseField name="observaciones" type="String">
Optional notes or observations (max 50 characters)
</ResponseField>

<ResponseField name="esPrueba" type="Boolean" required>
Indicates if this is a trial class
</ResponseField>

<ResponseField name="diaHoraCompleto" type="LocalDateTime">
Computed field combining date and time
</ResponseField>

## Get Class by ID

<api method="GET" endpoint="/api/v1/clases/{id}" />

Retrieves a single class by its ID without related entity details.

### Path Parameters

<ParamField path="id" type="Long" required>
The unique identifier of the class
</ParamField>

### Response

Returns a `Clase` object with the same fields as listed above.

</ResponseField>

Class date

<ResponseField name="hora" type="LocalTime" required>
Class time
</ResponseField>

<ResponseField name="duracion" type="Integer" required>

</ResponseField>

<ResponseField name="estado" type="Estado" required>

<ResponseField name="observaciones" type="String">
Optional observations
</ResponseField>

<ResponseField name="esPrueba" type="Boolean" required>
Trial class flag
</ResponseField>

<ResponseField name="instructorId" type="Long">
Instructor's unique identifier
</ResponseField>

Trial person's first name
</ResponseField>

<ResponseField name="personaPruebaApellido" type="String">
Trial person's last name
</ResponseField>

<ResponseField name="personaPruebaNombreCompleto" type="String">
Trial person's full name
</ResponseField>

<ResponseField name="caballoId" type="Long">
Horse's unique identifier
</ResponseField>

<ResponseField name="caballoNombre" type="String">
Horse's name
</ResponseField>

<ResponseField name="caballoTipo" type="String">
Horse's type
</ResponseField>

Combined date and time
</ResponseField>

## Get Class with Details by ID

<api method="GET" endpoint="/api/v1/clases/{id}/detalles" />

Retrieves a single class with complete details about instructor, student/trial person, and horse.

### Path Parameters

<ParamField path="id" type="Long" required>
The unique identifier of the class
</ParamField>

### Response

Returns a `ClaseResponseDto` object with all fields listed in the "List All Classes with Details" section.

### Error Responses

<ResponseField name="error" type="String">
Returns 404 Not Found if class does not exist
</ResponseField>

## Get Classes by State with Details

<api method="GET" endpoint="/api/v1/clases/estado/{estado}/detalles" />

Retrieves all classes with a specific state with complete details.

### Path Parameters

<ParamField path="estado" type="Estado" required>
The class state to filter by. Valid values:
- `PROGRAMADA` - Scheduled
- `INICIADA` - Started  
- `COMPLETADA` - Completed
- `CANCELADA` - Cancelled
- `ACA` - Absent with notice
- `ASA` - Absent without notice
</ParamField>

### Response

Returns an array of `ClaseResponseDto` objects matching the specified state.

### Error Responses

<ResponseField name="mensaje" type="String">
Returns 404 Not Found with message "No existen clases {estado}" if no classes exist with that state
</ResponseField>
