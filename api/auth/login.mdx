---
title: "Login"
description: "Authenticate a user and retrieve their account information"
api: "POST /api/v1/auth/login"
---

## Authentication

This endpoint uses HTTP Basic Authentication. Include the user's credentials in the Authorization header.

```
Authorization: Basic <base64(username:password)>
```

## Request

No request body is required. Authentication credentials are provided via the Basic Auth header.

## Response

<ResponseField name="id" type="long" required>
  Unique identifier for the user
</ResponseField>

<ResponseField name="username" type="string" required>
  User's unique username (max 50 characters)
</ResponseField>

<ResponseField name="email" type="string" required>
  User's email address (max 100 characters)
</ResponseField>

<ResponseField name="password" type="string" required>
  User's BCrypt-encrypted password
</ResponseField>

<ResponseField name="rol" type="RolSeguridad" required>
  User's role in the system. Possible values:
  - `ADMIN` - Full system management access
  - `INSTRUCTOR` - Manages their own classes
  - `ALUMNO` - Views their own classes
</ResponseField>

<ResponseField name="activo" type="boolean" required>
  Whether the user account is active (default: true)
</ResponseField>

<ResponseField name="fechaCreacion" type="LocalDateTime" required>
  Timestamp when the user account was created
</ResponseField>

<ResponseField name="personaDni" type="integer">
  DNI of the associated Alumno or Instructor (optional)
</ResponseField>

<ResponseField name="personaTipo" type="string">
  Type of associated person ("ALUMNO" or "INSTRUCTOR"). Max 20 characters.
</ResponseField>

<ResponseField name="avatarUrl" type="string">
  URL of the user's avatar image (max 255 characters)
</ResponseField>

## Error Responses

<ResponseField name="401 Unauthorized">
  Returned when credentials are invalid or missing
  ```json
  {
    "message": "Credenciales inválidas"
  }
  ```
</ResponseField>

<ResponseField name="404 Not Found">
  Returned when the authenticated username doesn't exist in the database
  ```json
  {
    "message": "Usuario not found with username: {username}"
  }
  ```
</ResponseField>

## Example

<CodeGroup>

```bash cURL
curl -X POST https://api.example.com/api/v1/auth/login \
  -H "Authorization: Basic $(echo -n 'username:password' | base64)"
```

```javascript JavaScript
const username = 'johndoe';
const password = 'secretpassword';
const credentials = btoa(`${username}:${password}`);

fetch('https://api.example.com/api/v1/auth/login', {
  method: 'POST',
  headers: {
    'Authorization': `Basic ${credentials}`
  }
})
  .then(response => response.json())
  .then(data => console.log(data));
```

```python Python
import requests
import base64

username = 'johndoe'
password = 'secretpassword'
credentials = base64.b64encode(f'{username}:{password}'.encode()).decode()

response = requests.post(
    'https://api.example.com/api/v1/auth/login',
    headers={'Authorization': f'Basic {credentials}'}
)

print(response.json())
```

</CodeGroup>
